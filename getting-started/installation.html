Installation

CommonJS Module Bundlers

Fluxbox is distributed on npm. You can install it with

npm install [--save] fluxbox

If you're using a client-side module bundler like Browserify or Webpack, you're done! Simply require("fluxbox") to get a reference to the library.

Browser Builds

Browser builds and corresponding source map files can be found on the Fluxbox releases page. Browser builds use a universal module definition, and should work in the following environments:

CommonJS

var Fluxbox = require("path/to/fluxbox");

AMD

define("someModule", ["Fluxbox"], function(Fluxbox) {
  // ...
});

Standalone

<script src="path/to/fluxbox.js"></script>

window.Fluxbox.createStore({ ... });
